# Spacing

## Supported scale tokens

当前 spacing scale 只支持这些 token：0, px, 1, 2, 3, 4, 5, 6, 7, 8, 10, 12, 16, 20, 24, 32, 40, 48, 56, 64。

## Padding

支持 p-*, px-*, py-*, pt-*, pr-*, pb-*, pl-*，其中 * 只能是上面的 scale token。

## Margin

支持 m-*, mx-*, my-*, mt-*, mr-*, mb-*, ml-*，其中 * 只能是上面的 scale token。

额外支持：

- mx-auto
- my-auto

## Gap

支持 gap-*, gap-x-*, gap-y-*，其中 * 只能是上面的 scale token。

## Current boundaries

- spacing 系列当前不支持负值。
- spacing 系列当前不支持任意值和自定义属性。
- top/right/bottom/left 偏移虽然也复用同一组 scale token，但属于定位 offset，不计入本页 spacing utility 集合。
