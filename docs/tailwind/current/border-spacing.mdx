# Border Spacing

border-spacing-*、border-spacing-x-*、border-spacing-y-* 当前都不参与静态画布预览。

## Current status

- parser 里没有对应的 ParsedStyle 字段，也没有解析分支。
- 这些 token 当前会落到未匹配路径，被标记为 Unsupported。
- 原始 class 字符串仍会按导入/导出边界文档中的约定保留，用于 round-trip export。

## What to use instead

如果目标是当前画布上的可见间距，请使用已支持的 gap-*、gap-x-*、gap-y-*、p-*、m-* 系列 utility。
