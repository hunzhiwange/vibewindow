# Border Color

当前静态画布只支持基础 border-color token 形式的颜色 utility。事实来源是 parse_color_utility 和 TailwindColors::BORDER_COLOR_TOKENS。

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| border-white | border-color: #ffffff |
| border-black | border-color: #000000 |
| border-transparent | border-color: transparent |
| border-gray-100 | border-color: #f3f4f6 |
| border-gray-500 | border-color: #6b7280 |
| border-gray-600 | border-color: #70808f |
| border-gray-800 | border-color: #1f2937 |
| border-red-500 | border-color: #ef4444 |
| border-blue-100 | border-color: #dbeafe |
| border-blue-500 | border-color: #3b82f6 |
| border-green-500 | border-color: #22c55e |
| border-yellow-500 | border-color: #eab308 |
| border-indigo-500 | border-color: #6366f1 |
| border-indigo-600 | border-color: #4f46e5 |
| border-sky-500 | border-color: #0ea5e9 |

## Current boundaries

- 仅支持整框颜色，即 border 加受支持颜色 token 的组合。
- border-t-indigo-500、border-x-indigo-500、divide-indigo-500 这类带方向或 divide 的颜色 utility 当前没有 parser 分支。
- 透明度后缀会被压平到基础颜色 token，例如 border-indigo-500/50 在当前画布上按 border-indigo-500 处理，不保留 alpha。
- 自定义属性和任意值颜色当前不进入静态画布预览。
