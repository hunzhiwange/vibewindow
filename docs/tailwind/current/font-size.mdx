# Font Size

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| text-sm | font-size: 14px |
| text-base | font-size: 16px |
| text-lg | font-size: 18px |
| text-xl | font-size: 20px |
| text-2xl | font-size: 24px |
| text-3xl | font-size: 30px |

## Current boundaries

- 当前 parser 只支持这 6 个显式字体尺寸 token。
- 没有 text-xs、text-4xl 及更大尺寸的 parser 分支。
- 当前没有独立的 line-height 联动语法，例如 text-sm/6。
