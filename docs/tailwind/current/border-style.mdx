# Border Style

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| border-solid | border-style: solid |
| border-dashed | border-style: dashed |
| border-dotted | border-style: dotted |
| border-double | border-style: double |
| border-hidden | border-style: hidden |
| border-none | border-style: none |

## Current boundaries

- 当前只支持整框 border-* 样式，不支持 divide-dashed、divide-dotted 这类 divider style utility。
- 也没有单侧 border style 的独立 parser 分支。
- 这些样式只影响静态边框绘制，不引入浏览器级别的复杂合成语义。
