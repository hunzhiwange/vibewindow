# Current Tailwind Coverage

本目录记录的是当前仓库真实支持的静态画布 Tailwind 子集，不是上游 Tailwind 的完整文档镜像。

## Covered families

- size and constraints: `w-*`、`h-*`、`w-full`、`h-full`、`w-auto`、`h-auto`、`max-w-screen-*`，以及有限的 arbitrary px size
- spacing: padding、margin、gap 及 mx-auto、my-auto
- display: block、inline、inline-block、flex、inline-flex、grid、hidden
- position and offsets: `relative`、`absolute` 以及 `top/right/bottom/left-*` 的 spacing scale 与有限 arbitrary px 偏移
- flex/grid 基础: direction、items、justify、grow、shrink、basis、grid-cols-1|2|3|4|5|6|12
- colors: background-color、text-color、border-color 的有限 token 集合
- typography: font-size、font-weight、text-align、font-style、text-decoration、text-transform、letter-spacing、line-height
- border: width、style、radius、divide width 和 reverse
- effects and transforms: opacity、translate-x/y、outline、shadow-none|shadow-sm|shadow|shadow-md|shadow-lg

## Explicitly outside this subset

- 运行时响应式布局和交互态只会被压平成静态快照，见 canvas-support-boundaries.mdx
- `rotate`、`scale`、`skew` 以及更复杂的 transform 当前不是静态画布真实渲染能力
- dark variant、animation、transition、filter、backdrop、ring、复杂 shadow 当前不是静态画布真实渲染能力
- 本目录若新增或删除某个 utility 页面，应同时核对 parser/mod.rs 和 colors.rs 中的事实来源
