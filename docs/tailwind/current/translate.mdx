## Canvas support

The current canvas preview supports only spacing-scale based 2D `translate-x-*` and `translate-y-*` utilities.

`rotate-*`, `scale-*`, and `skew-*` remain explicitly unsupported, because they would require non-axis-aligned geometry to keep rendering, bounds, and hit-testing consistent.
