# Text Align

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| text-left | text-align: left |
| text-center | text-align: center |
| text-right | text-align: right |
| text-justify | text-align: justify |
| text-start | mapped to left in parser |
| text-end | mapped to right in parser |

## Current boundaries

- 当前 parser 把 text-start 归一成 left，把 text-end 归一成 right；它们不是保留逻辑方向的独立值。
- 这些 utility 只影响静态文本排版快照，不代表运行时双向排版语义完整实现。
