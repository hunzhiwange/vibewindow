# Line Height

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| leading-none | line-height: 1.0 |
| leading-tight | line-height: 1.25 |
| leading-snug | line-height: 1.375 |
| leading-normal | line-height: 1.5 |
| leading-relaxed | line-height: 1.625 |
| leading-loose | line-height: 2.0 |

## Current boundaries

- 当前只支持这些命名行高 token。
- 没有 leading-3、leading-[...]、leading-(...) 这类数值或任意值 parser 分支。
