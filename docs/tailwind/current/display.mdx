# Display

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| block | display: block |
| inline-block | display: inline-block |
| inline | display: inline |
| flex | display: flex |
| inline-flex | display: inline-flex |
| grid | display: grid |
| hidden | display: none |

## Current boundaries

- 当前没有独立的 inline-grid、contents、table、flow-root 等 display parser 分支。
- hidden 直接在静态画布上折叠为不可见快照，不保留浏览器运行时交互语义。
