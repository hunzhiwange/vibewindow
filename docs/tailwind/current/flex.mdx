# Flex

## Container direction

| Utility | Canvas behavior |
| --- | --- |
| flex-row | flex-direction: row |
| flex-row-reverse | flex-direction: row-reverse |
| flex-col | flex-direction: column |
| flex-col-reverse | flex-direction: column-reverse |

## Cross-axis alignment

| Utility | Canvas behavior |
| --- | --- |
| items-start | align-items: flex-start |
| items-center | align-items: center |
| items-end | align-items: flex-end |

## Main-axis distribution

| Utility | Canvas behavior |
| --- | --- |
| justify-start | justify-content: flex-start |
| justify-center | justify-content: center |
| justify-end | justify-content: flex-end |
| justify-between | justify-content: space-between |

## Flex item sizing

| Utility | Canvas behavior |
| --- | --- |
| grow | flex-grow: 1 |
| grow-0 | flex-grow: 0 |
| grow-N | flex-grow: parsed number |
| shrink | flex-shrink: 1 |
| shrink-0 | flex-shrink: 0 |
| shrink-N | flex-shrink: parsed number |
| flex-1 | grow 1, shrink 1, basis 0 |
| flex-auto | grow 1, shrink 1 |
| flex-none | grow 0, shrink 0 |
| basis-* | flex-basis from the supported spacing scale |

## Current boundaries

- 当前没有 justify-around、justify-evenly、items-stretch、self-*、content-* 等 parser 分支。
- basis-* 只支持 spacing scale token，不支持百分比、自定义属性和任意值。
- flex-wrap、order、place-* 当前不进入静态画布样式。
