# Border Width

## Whole border

| Utility | Canvas behavior |
| --- | --- |
| border | all sides 1px |
| border-0 | all sides 0px |
| border-2 | all sides 2px |
| border-4 | all sides 4px |
| border-8 | all sides 8px |

## Individual sides

| Utility family | Supported suffixes |
| --- | --- |
| border-t-* | border-t, border-t-0, border-t-2, border-t-4, border-t-8 |
| border-r-* | border-r, border-r-0, border-r-2, border-r-4, border-r-8 |
| border-b-* | border-b, border-b-0, border-b-2, border-b-4, border-b-8 |
| border-l-* | border-l, border-l-0, border-l-2, border-l-4, border-l-8 |

## Axis and logical borders

| Utility | Canvas behavior |
| --- | --- |
| border-x | left and right 1px |
| border-x-2 | left and right 2px |
| border-x-4 | left and right 4px |
| border-y | top and bottom 1px |
| border-y-2 | top and bottom 2px |
| border-y-4 | top and bottom 4px |
| border-s | inline start 1px |
| border-s-2 | inline start 2px |
| border-s-4 | inline start 4px |
| border-e | inline end 1px |
| border-e-2 | inline end 2px |
| border-e-4 | inline end 4px |

## Divide utilities

| Utility | Canvas behavior |
| --- | --- |
| divide-x, divide-x-2, divide-x-4 | horizontal child divider width |
| divide-y, divide-y-2, divide-y-4 | vertical child divider width |
| divide-x-reverse | reverse horizontal divider direction flag |
| divide-y-reverse | reverse vertical divider direction flag |

## Current boundaries

- 自定义属性和任意值 border width 当前不支持。
- border-x-8、border-y-8、border-s-8、divide-x-8 等未在 parser 显式列出的 token 当前不支持。
- divide 目前只记录宽度和 reverse 标记，不包含独立的 divide style 或 divide color parser 分支。
