# Font Weight

## Supported utilities

| Utility | Canvas behavior |
| --- | --- |
| font-light | font-weight: 300 |
| font-normal | font-weight: 400 |
| font-semibold | font-weight: 600 |
| font-bold | font-weight: 700 |

## Current boundaries

- 当前没有 font-medium、font-extrabold、font-black 等 parser 分支。
- 这些权重只影响静态文本绘制和测量，不代表完整浏览器字体回退链行为。
