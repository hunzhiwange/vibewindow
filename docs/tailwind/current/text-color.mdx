## Current boundaries

- 当前 text-* 颜色和字体大小共用前缀，parser 会先尝试解析颜色，再退回到显式字体大小 token。
- 因此只有这一小组颜色 token 会作为 text color 生效，其他 text-* token 不会被自动视为颜色。
- 自定义属性、任意值和独立 alpha 文本颜色当前不进入静态画布预览。
