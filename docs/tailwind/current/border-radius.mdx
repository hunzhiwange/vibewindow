# Border Radius

当前静态画布只支持整体圆角半径，不支持按边、按角或逻辑方向拆分的圆角 utility。

## Supported utilities

| Utility | Radius |
| --- | --- |
| rounded-none | 0px |
| rounded-xs | 2px |
| rounded | 4px |
| rounded-sm | 4px |
| rounded-md | 6px |
| rounded-lg | 8px |
| rounded-xl | 12px |
| rounded-2xl | 16px |
| rounded-3xl | 24px |
| rounded-4xl | 32px |
| rounded-full | 9999px |

## Current boundaries

- rounded-t-*、rounded-r-*、rounded-bl-*、rounded-s-* 等分侧和逻辑圆角当前没有 parser 分支。
- 自定义属性和任意值圆角当前不进入静态画布预览。
- 当前 rounded 与 rounded-sm 都映射到 4px，这是 parser 中的实际行为。
