# 技能设计的说服原则

## 概述

LLM 对与人类相同的说服原则做出反应。理解这种心理学有助于你设计更有效的技能 - 不是为了操纵，而是为了确保即使在压力下也遵循关键实践。

**研究基础：**Meincke 等人（2025）测试了7种说服原则，N=28,000次 AI 对话。说服技术使合规率提高了一倍以上（33% → 72%，p < .001）。

## 七大原则

### 1. 权威
**它是什么：**对专业知识、凭证或官方来源的顺从。

**在技能中如何工作：**
- 强制性语言："你必须"、"绝不"、"总是"
- 不可协商的框架："无例外"
- 消除决策疲劳和合理化

**何时使用：**
- 强制执行纪律的技能（TDD、验证要求）
- 安全关键的实践
- 既定的最佳实践

**示例：**
```markdown
✅ 在测试之前编写代码？删除它。重新开始。无例外。
❌ 在可行时考虑先编写测试。
```

### 2. 承诺
**它是什么：**与先前的行动、声明或公开声明的一致性。

**在技能中如何工作：**
- 要求宣布："宣布技能使用"
- 强制明确选择："选择 A、B 或 C"
- 使用跟踪：TodoWrite 用于检查清单

**何时使用：**
- 确保技能实际被遵循
- 多步骤流程
- 问责机制

**示例：**
```markdown
✅ 当你找到技能时，你必须宣布："我正在使用[技能名称]"
❌ 考虑让你的伙伴知道你正在使用哪个技能。
```

### 3. 稀缺性
**它是什么：**来自时间限制或有限可用性的紧迫感。

**在技能中如何工作：**
- 时间限制要求："在继续之前"
- 顺序依赖："在 X 之后立即"
- 防止拖延

**何时使用：**
- 即时验证要求
- 时间敏感的工作流程
- 防止"我稍后再做"

**示例：**
```markdown
✅ 完成任务后，立即在继续之前请求代码审查。
❌ 你可以在方便的时候审查代码。
```

### 4. 社会认同
**它是什么：**与他人所做的或被认为是正常的行为保持一致。

**在技能中如何工作：**
- 通用模式："每次"、"总是"
- 失败模式："没有 Y 的 X = 失败"
- 建立规范

**何时使用：**
- 记录通用实践
- 警告常见失败
- 加强标准

**示例：**
```markdown
✅ 没有 TodoWrite 跟踪的检查清单 = 步骤被跳过。每次。
❌ 有些人发现 TodoWrite 对检查清单有帮助。
```

### 5. 团结
**它是什么：**共同身份、"我们性"、群体归属感。

**在技能中如何工作：**
- 协作语言："我们的代码库"、"我们是同事"
- 共同目标："我们都想要质量"

**何时使用：**
- 协作工作流程
- 建立团队文化
- 非层次化实践

**示例：**
```markdown
✅ 我们是一起工作的同事。我需要你的诚实技术判断。
❌ 如果我错了，你应该可能告诉我。
```

### 6. 互惠
**它是什么：**回报所获得好处的义务。

**如何工作：**
- 谨慎使用 - 可能感觉操纵性
- 技能中很少需要

**何时避免：**
- 几乎总是（其他原则更有效）

### 7. 喜好
**它是什么：**偏好与我们喜欢的人合作。

**如何工作：**
- **不要用于合规**
- 与诚实反馈文化冲突
- 创造阿谀奉承

**何时避免：**
- 始终用于强制执行纪律

## 按技能类型的原则组合

| 技能类型 | 使用 | 避免 |
|------------|-----|-------|
| 强制执行纪律 | 权威 + 承诺 + 社会认同 | 喜好、互惠 |
| 指导/技术 | 适度权威 + 团结 | 重度权威 |
| 协作 | 团结 + 承诺 | 权威、喜好 |
| 参考 | 仅清晰度 | 所有说服 |

## 为什么有效：心理学

**清晰的规则减少合理化：**
- "你必须"消除决策疲劳
- 绝对语言消除"这是一个例外吗？"问题
- 明确的反合理化反制措施关闭特定漏洞

**实施意图创造自动行为：**
- 清晰的触发器 + 所需操作 = 自动执行
- "当 X，做 Y"比"通常做 Y"更有效
- 减少合规的认知负担

**LLM 是类人的：**
- 在包含这些模式的人类文本上训练
- 权威语言在训练数据中先于合规
- 承诺序列（声明 → 行动）经常被建模
- 社会认同模式（每个人都做 X）建立规范

## 伦理使用

**合法：**
- 确保遵循关键实践
- 创建有效的文档
- 防止可预测的失败

**非法：**
- 为了个人利益而操纵
- 创建虚假的紧迫感
- 基于内疚的合规

**测试：**如果用户完全理解，这种技术是否服务于用户的真正利益？

## 研究引用

**Cialdini, R. B. (2021).** *Influence: The Psychology of Persuasion (New and Expanded).* Harper Business.
- 说服的七项原则
- 影响研究的实证基础

**Meincke, L., Shapiro, D., Duckworth, A. L., Mollick, E., Mollick, L., & Cialdini, R. (2025).** Call Me A Jerk: Persuading AI to Comply with Objectionable Requests. University of Pennsylvania.
- 在 N=28,000 次 LLM 对话中测试了7项原则
- 使用说服技术合规率从33%增加到72%
- 权威、承诺、稀缺性最有效
- 验证 LLM 行为的类人模型

## 快速参考

在设计技能时，问：

1. **它是什么类型？**（纪律与指导与参考）
2. **我试图改变什么行为？**
3. **哪些原则适用？**（通常是权威 + 承诺用于纪律）
4. **我是否结合了太多？**（不要使用全部七个）
5. **这是伦理的吗？**（服务于用户的真正利益？）
