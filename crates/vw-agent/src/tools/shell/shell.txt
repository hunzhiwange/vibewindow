执行 Shell 命令。在指定工作目录中运行命令，支持超时控制和沙箱隔离。

参数：
- `command`（必填）：要执行的命令
- `description`（必填）：用 5-10 个词描述命令作用
- `timeout`（可选）：超时毫秒数，默认 120000
- `workdir`（可选）：工作目录，推荐使用此参数替代 `cd`
- `approved`（可选）：监督模式下显式批准命令

安全限制：
- 命令在工作目录沙箱中执行
- 只能访问允许的路径
- 环境变量经过白名单过滤
- 危险命令需要用户确认

推荐用法：
- 用 Glob/Grep 工具替代 `find`/`grep` 命令
- 用 FileEdit 工具替代 `sed`/`awk` 命令
- 用 FileRead 工具替代 `cat`/`head`/`tail` 命令
- 用 FileWrite 工具替代 `echo > file` 命令
